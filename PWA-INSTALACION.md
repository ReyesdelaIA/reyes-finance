# Instalar Reyes Finance en tu iPhone

Tu dashboard ya está configurado como **PWA** (Progressive Web App). Puedes instalarlo en la pantalla de inicio de tu iPhone como si fuera una app nativa.

## Pasos para instalar en iPhone

1. Abre **Safari** en tu iPhone (no Chrome u otros navegadores).
2. Ve a: **https://reyes-finance.vercel.app**
3. Inicia sesión con Google si es necesario.
4. Toca el botón **Compartir** (cuadrado con flecha hacia arriba).
5. Desplázate y toca **"Agregar a pantalla de inicio"**.
6. Confirma el nombre (por ejemplo "Reyes Finance") y toca **"Agregar"**.

Listo. Verás el ícono en tu pantalla de inicio y podrás abrirlo como cualquier app.

## Notas

- Funciona mejor en Safari (iOS requiere Safari para PWA).
- Se abrirá a pantalla completa, sin barra del navegador.
- Necesitas conexión a internet para usar el dashboard.
- Los datos se actualizan al hacer push en GitHub (igual que la versión web).
